# 🎈 Blank app template

A simple Streamlit app template for you to modify!

> 📈 **New:** this repo also contains a [Telegram stock alert bot](stock_alert_bot/README.md)
> that sends real-time technical-analysis buy/sell alerts. See its README for setup;
> quick start: `pip install -r requirements.txt`, set `TELEGRAM_BOT_TOKEN`, then
> `python -m stock_alert_bot`.

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://blank-app-template.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
